# History

## 2026-06-24
### 주요 변경 및 추가 사항

1. **연장근로 현황 통합 장부 기능 도입**
   - 기존에는 '발생/만료 관리' 표와 '사용 내역' 표가 분리되어 있었으나, 이를 하나의 통합된 장부 형태(Ledger)의 표로 개편.
   - 계좌 입출금 내역처럼 일자별 발생, 사용, 소멸, 잔여 연장근로 시간 흐름을 연단위로 한 눈에 직관적으로 볼 수 있게 수정.

2. **연장 승인 기능 직관성 개선**
   - 연장근로 시간의 유효기간 연장 버튼이 발생 일자가 아닌, **실제 소멸 예정일 행의 우측**에 표시되도록 변경.
   - 관리자가 연장 버튼을 클릭할 때 뜨는 모달 창에 **'새로운 만료 예정일'** 을 지정할 수 있는 Date Picker(날짜 선택창) 추가 (기본값으로 기존 소멸일 + 2개월 지정).
   - 이미 연장된 이력이 있는 시간은 연장 버튼 대신 "연장됨" 텍스트가 표시되도록 보완.

3. **환경설정 및 비밀번호 변경 기능 추가**
   - 프로그램 좌측 하단 사이드바에 톱니바퀴 모양의 **[⚙️ 설정]** 탭 신설.
   - 기존 관리자 비밀번호를 입력 후 새로운 비밀번호로 변경할 수 있는 보안 편의 기능 추가 (기본 비밀번호: `주형진에게 문의`).

4. **버그 수정 및 UI 보완**
   - **대시보드 주간 근무표 표출 오류 수정**: UI 요소 참조 오류를 수정하여 주간 근무표가 대시보드에 정상적으로 뜨도록 해결.
   - **월간 근무표 UI 오류 수정**: 월간 엑셀 다운로드 화면 하단 월간 근무표의 맨 우측 **'잔여연장근로'** 컬럼 제목(Header) 누락 현상 수정 완료.
   - **자바스크립트 Syntax 오류 복구**: `app.js` 내부에 괄호 누락으로 인해 전체 버튼이 먹통되던 크리티컬 버그 수정.
   - **날짜 입력기 요일 표시 정렬**: 날짜 입력칸과 우측 요일이 동일 선상에 오도록 UI Flex 레이아웃 보완.
   - **휴가류 선택 시 연장근로 입력 숨김**: 정상근무/반차/휴무 외의 휴가류(공휴일 등) 선택 시 연장근로 시간 입력칸이 자동으로 숨겨지도록 UI 로직 보완.

5. **연장근로 로직 및 엑셀 다운로드 개선**
   - **월 20시간(1200분) 1.5배수 로직 유지**: 월 누적 20시간 이하는 1배수, 초과는 1.5배수로 가산 적립되는 룰을 완벽히 유지.
   - **30분 절삭**: 40분 -> 30분, 1시간 10분 -> 1시간 등으로 30분 단위 버림 처리되도록 산술 최적화 완료.
   - **연간 근무표 엑셀 잔여 연장근로**: 빈 데이터(연말) 구간에서 잔여 연장근로가 0이 되는 현상 수정
   - **엑셀 다운로드 가독성 컬러 적용**: 공휴일/휴가류(빨간색), 반차류(초록색), 연장근로휴무(파란색) 자동 폰트 컬러 적용.
